SIRCLO Template Development Kit

Getting started to SIRCLO TDK.

Prerequisite:

  1. Download SMARTY from its official website [--Requirement-- Smarty v3.1.21 http://www.smarty.net/download]
  2. Apache or Nginx as webserver

Preparation step:

  1. Put this tdk directory into your working development. E.g /Users/Project/tdk
  2. Set up your virtual host
  3. Edit config.ini to set absolute path of SMARTY location and the webserver you use
  4. From terminal, go to tdk directory and run ./tdk_init.sh. The script will generate new local host for your access to your template list.
